# JoinQuran Website Commands

This file contains the necessary commands to set up and run the JoinQuran website project.

## Prerequisites

*   Node.js (v18 or later)
*   npm

## Installation

1.  Navigate to the `app` directory:
    ```bash
    cd app
    ```
2.  Install the dependencies:
    ```bash
    npm install
    ```

## Running the Development Server

1.  Navigate to the `app` directory:
    ```bash
    cd app
    ```
2.  Run the development server:
    ```bash
    npm run dev
    ```
3.  Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Building for Production

1.  Navigate to the `app` directory:
    ```bash
    cd app
    ```
2.  Build the project:
    ```bash
    npm run build
    ```
